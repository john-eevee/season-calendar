# Season Calendar

A beautiful single-page app that displays the current date as "days of the season" (e.g., "37th of Summer"), automatically detecting your hemisphere based on locale.

## Features

- Automatic hemisphere detection (Northern/Southern) based on your locale
- Current season display with day count
- Beautiful Catppuccin color scheme
- Smooth animations and transitions
- Progress bar showing season completion
- Fully responsive design

## Deployment

This site is ready for GitHub Pages deployment:

1. Push this repository to GitHub
2. Enable GitHub Pages in repository settings
3. Select the main branch as source
4. Your site will be live at `https://yourusername.github.io/season-calendar/`
